# 二轮充电前端/小程序接口说明 v2

本文档面向后台前端和小程序对接人员。二轮充电统一使用 `/twoWheel/*`，小程序接口统一使用 `/twoWheel/xcx/*`。BKV 只是设备品牌，通过 `device_brand='BKV'` 区分；一个站点下可以有多个品牌的桩，实际设备品牌以桩和枪的 `device_brand` 为准。

基础路径：

```text
https://fengzecj.genchuan.cn:60000/parking/service
```

## 1. 通用说明

所有接口均为 `POST`，请求体为 JSON。

成功响应：

```json
{
  "success": true,
  "data": {}
}
```

失败响应：

```json
{
  "success": false,
  "error": "错误原因"
}
```

列表响应：

```json
{
  "success": true,
  "data": {
    "count": 1,
    "rows": []
  }
}
```

列表通用请求字段：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `page` | number | 否 | 页码，默认 `1` |
| `page_size` | number | 否 | 每页数量，默认 `20`，最大 `200` |
| `data_owner_id` | string | 否 | 运营方 ID |
| `status` | string | 否 | 状态 |
| `device_brand` | string | 否 | 设备品牌 |

支持模糊查询的字段：`name`、`station_no`、`pile_no`、`port_no`、`device_no`、`order_no`、`card_no`、`mobile`。

`update` 接口不是只能改名；除 `id` 外，可提交该表支持的任意业务字段，后端按传入字段更新，未传字段保持原值。

## 2. 数据库表项

所有二轮表都有：

| 字段 | 说明 |
| --- | --- |
| `id` | 主键 ID |
| `created_at` | 创建时间 |
| `updated_at` | 更新时间 |
| `deleted_at` | 删除时间 |
| `deleted` | 软删除标记 |

建表脚本：`二轮充电文档/二轮充电独立模块建表SQL.sql`

### 2.1 设备表

| 表名 | 说明 | 关键字段 |
| --- | --- | --- |
| `two_wheel_station` | 二轮充电站 | `id`、`station_no`、`two_wheel_station_no`、`name`、`data_owner_id`、`parking_lot_id`、`address`、`longitude`、`latitude`、`province`、`city`、`district`、`property_unit`、`property_contact`、`property_phone`、`service_phone`、`business_hours`、`status`、`device_brand`、`two_wheel_strategy_ids`、`operator_id`、`invoice_apply`、`invoice_info`、`whitelist_apply`、`push_status`、`push_time`、`need_push_map`、`options` |

| `two_wheel_port` | 二轮枪口 | `id`、`port_no`、`device_no`、`two_wheel_pile_id`、`two_wheel_station_id`、`data_owner_id`、`device_brand`、`port_status`、`using_status`、`max_power`、`operator_id`、`parking_space_arr`、`voltage`、`voltage_upper_limits`、`voltage_lower_limits`、`current`、`qrcode_id`、`last_status_at`、`options` |

### 3.1 `POST /twoWheel/station/create`

用途：创建二轮站点。

请求：

```json
{
  "station_no": "TW-ST-001",
  "two_wheel_station_no": "TW-ST-001",
  "name": "丰泽二轮站",
  "data_owner_id": "owner-id",
  "parking_lot_id": "lot-id",
  "address": "站点地址",
  "longitude": 119.12345678,
  "latitude": 26.12345678,
  "province": "福建省",
  "city": "福州市",
  "district": "丰泽区",
  "service_phone": "0591-00000000",
  "business_hours": "00:00-24:00",
  "device_brand": "BKV",
  "status": "enable",
  "two_wheel_strategy_ids": {
    "duration": "strategy-duration-id",
    "electricity_service": "strategy-energy-id",
    "per_use": "strategy-per-use-id"
  },
  "options": {}
}
```

响应：

```json
{
  "success": true,
  "data": {
    "id": "station-id",
    "station_no": "TW-ST-001",
    "two_wheel_station_no": "TW-ST-001",
    "name": "丰泽二轮站",
    "status": "enable",
    "device_brand": "BKV"
  }
}
```
